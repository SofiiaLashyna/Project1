Detailed Description

template<typename T>
class Graph< T >

An abstract base class defining the interface for a graph structure.

This class provides the core storage (vertices, edges) and a common interface that concrete classes (GraphList, GraphMatrix) must implement.

Template Parameters
TThe type of data stored in the vertices.
