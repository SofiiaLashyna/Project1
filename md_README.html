ï»¿### Galaxy Simulation Project

Galaxy Simulation is a C++ programming educational project that models the structure of a galaxy using graphs. Each astronomical object (star system, nebula, etc.) is represented as a vertex of a graph, and the connections between them are represented as edges with weights (e.g., distance).

Key Features

This project demonstrates a wide range of C++ and OOP concepts:

  • Object-Oriented Modeling: A clear inheritance hierarchy (CelestialObject -> Star, Planet, Nebula).
  • Polymorphism: Use of virtual functions for dynamic behavior (like displayInfo() and getType()).
  • Advanced Memory Management: Correct use of pointers, references, and RAII principles (e.g., custom destructors in Galaxy and StarSystem to prevent memory leaks).
  • Generic Programming: Extensive use of C++ templates (Graph, Galaxy, Queue).
  • Graph Algorithms:
    • Adjacency List and Adjacency Matrix implementations.
    • Traversal algorithms (BFS, DFS).
    • Pathfinding (Dijkstra).
  • Unit Testing: A suite of tests (using Google Test) to verify core logic.
  • GUI: A graphical interface built with Qt to visualize the generated galaxy.

Technologies Used

  • C++20
