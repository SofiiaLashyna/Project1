Detailed Description

template<typename GraphType, typename T>
class GraphAlgorithm< GraphType, T >

Abstract interface for graph algorithms (Strategy Pattern).

Concrete strategies must implement run() to perform the selected algorithm (e.g., BFS, DFS, Dijkstra).

Template Parameters
GraphTypeGraph type used by the strategy.
TType stored in graph vertices.
