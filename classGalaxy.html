Detailed Description

template<typename GraphType>
class Galaxy< GraphType >

Manages all celestial objects and their relationships (graph) in a galaxy.

This class acts as the top-level container, owning all CelestialObject pointers (StarSystem, Nebula, etc.) and managing the graph that connects them.

Template Parameters
GraphTypeThe type of graph used to store object relationships (e.g., GraphList<CelestialObject*> or GraphMatrix<CelestialObject*>).
